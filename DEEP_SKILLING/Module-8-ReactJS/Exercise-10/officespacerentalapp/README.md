# Module 8 - Exercise 10

## Objective

Create a React application using JSX to display office space details.

## Concepts Covered

- JSX Elements
- JSX Attributes
- JSX Expressions
- React.createElement()
- Rendering JSX to DOM
- Inline CSS
- Conditional Styling
- map() Function

## Files Modified

- App.js
- App.css
- office.jpg

## Features

- Displays office image
- Displays office name, rent and address
- Renders multiple office records using map()
- Rent is displayed in:
  - Red if Rent ≤ 60000
  - Green if Rent > 60000

## Result

The Office Space Rental application successfully demonstrates JSX, inline CSS, object rendering, conditional styling, and list rendering using React.
