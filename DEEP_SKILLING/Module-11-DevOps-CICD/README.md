# Module 11 - DevOps and CI/CD

## Overview

This module introduces the fundamentals of DevOps, Continuous Integration (CI), Continuous Delivery (CD), and commonly used DevOps tools. It provides an understanding of how DevOps improves software development, testing, deployment, and collaboration between development and operations teams.

## Learning Objectives

After completing this module, I learned to:

- Explain the concept of DevOps.
- Understand Continuous Integration (CI) and Continuous Delivery (CD).
- Differentiate between CI and CD.
- Identify popular DevOps tools and their purposes.
- Understand the benefits of DevOps practices.

## Topics Covered

- Introduction to DevOps
- Goals and Benefits of DevOps
- Continuous Integration (CI)
- Continuous Delivery (CD)
- CI vs CD
- Popular DevOps Tools
- DevOps Workflow
