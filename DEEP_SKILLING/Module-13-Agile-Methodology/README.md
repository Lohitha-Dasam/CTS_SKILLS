# Module 13 - Agile Methodology

## Overview

This module introduces Agile methodology, its principles, Scrum framework, Agile estimation techniques, sprint planning, user stories, and acceptance criteria. It focuses on iterative development, collaboration, and delivering customer value through Agile best practices.

## Learning Objectives

After completing this module, I learned to:

- Explain Agile principles and the Agile Manifesto.
- Differentiate Agile and Waterfall methodologies.
- Understand Scrum roles, ceremonies, and artifacts.
- Learn Agile estimation techniques.
- Interpret velocity and burndown charts.
- Understand sprint planning.
- Write effective user stories.
- Define acceptance criteria using Given-When-Then.
- Understand Agile team collaboration.

## Topics Covered

- Introduction to Agile
- Agile Manifesto
- Agile vs Waterfall
- Scrum Framework
- Scrum Roles
- Scrum Ceremonies
- Scrum Artifacts
- Story Points
- Planning Poker
- Sprint Planning
- Velocity & Burndown Charts
- User Stories
- INVEST Principle
- Acceptance Criteria
