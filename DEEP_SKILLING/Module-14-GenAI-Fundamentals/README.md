# Module 14 - Generative AI Fundamentals

## Overview

This module introduces the fundamentals of Generative AI, Prompt Engineering, GitHub Copilot, AI-assisted software development, and the ethical and security considerations of using AI coding tools.

## Learning Objectives

After completing this module, I learned to:

- Explain the fundamentals and evolution of Generative AI.
- Differentiate between Generative AI and Traditional AI.
- Apply prompt engineering techniques.
- Follow prompt engineering best practices.
- Understand GitHub Copilot and its features.
- Configure GitHub Copilot in Visual Studio Code.
- Generate code, documentation, and test cases using AI.
- Refactor code using AI assistance.
- Understand security, privacy, and licensing risks of AI-generated code.
- Follow responsible AI usage practices.
