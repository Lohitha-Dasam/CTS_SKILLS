# Exercise 02 - Git Ignore

## Objective

Learn how to ignore unwanted files and folders using `.gitignore`.

## Tools Used

- Git
- Git Bash
- Visual Studio Code

## Commands Executed

```bash
git init
mkdir logs
touch app.log
touch logs/error.log
touch index.html
touch style.css
touch .gitignore
git status
git add .
git commit -m "Added gitignore demonstration"
git log
```

## Files Created

- app.log
- logs/error.log
- index.html
- style.css
- .gitignore
- Outputs/Lab10_Exercise02_Output.txt

## Result

Successfully ignored unwanted files and folders using `.gitignore`.
