# Exercise 05 - Push to Remote Repository

## Objective

Learn how to synchronize a local Git repository with a remote GitHub repository.

## Tools Used

- Git
- Git Bash
- GitHub
- Visual Studio Code

## Commands Executed

```bash
git status
git branch
git remote -v
git add .
git commit -m "Completed Module 10 Version Control exercises"
git pull origin main
git push origin main
```

- Outputs/Lab10_Exercise05_Output.txt

## Result

Successfully verified the repository status, synchronized the local repository with GitHub, and confirmed the latest changes were available in the remote repository.
