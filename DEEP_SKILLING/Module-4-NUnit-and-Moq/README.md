# Module 4 – NUnit and Moq

## Overview

This module focuses on automated unit testing using the NUnit framework and mocking external dependencies using the Moq framework.

## Learning Objectives

- Understand Automated Testing
- Understand Unit Testing
- Write NUnit Test Cases
- Use Assertions
- Test Exceptions
- Test Collections
- Use TestCase and TestCaseSource
- Mock External Dependencies
- Dependency Injection
- Constructor Injection
- Test Driven Development (TDD)
