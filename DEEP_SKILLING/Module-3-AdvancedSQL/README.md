# Module 3 - Advanced SQL Using SQL Server

## Overview

This module focuses on advanced SQL programming concepts using Microsoft SQL Server. It covers database programming, query optimization, stored procedures, user-defined functions, triggers, cursors, and exception handling.

## Exercises Completed

### Exercise 1 - Advanced SQL Concepts
- Window Functions
- ROW_NUMBER()
- RANK()
- DENSE_RANK()
- Common Table Expressions (CTE)

### Exercise 2 - Indexes
- Clustered Index
- Non-Clustered Index
- Composite Index
- Covering Index

### Exercise 3 - Views
- Simple Views
- Views with Joins
- Computed Columns
- Employee Report Views

### Exercise 4 - Stored Procedures
- Creating Stored Procedures
- Altering Procedures
- Dropping Procedures
- Parameters
- Output Parameters
- Transactions
- Dynamic SQL
- TRY...CATCH Error Handling

### Exercise 5 - User Defined Functions
- Scalar Functions
- Table-Valued Functions
- Nested Functions
- Function Modification
- Function Execution

### Exercise 6 - Triggers
- AFTER Trigger
- INSTEAD OF Trigger
- Trigger Modification
- Trigger Deletion
- Automatic Annual Salary Update

### Exercise 7 - Cursors
- Simple Cursor
- Static Cursor
- Dynamic Cursor
- Forward-Only Cursor
- Keyset Cursor

### Exercise 8 - Exception Handling
- TRY...CATCH
- THROW
- RAISERROR
- Nested Exception Handling
- Transactions
- Error Logging

## Technologies Used

- Microsoft SQL Server
- SQL Server Management Studio (SSMS)
- SQL (T-SQL)

## Learning Outcomes

- Developed advanced SQL queries.
- Implemented database objects such as Views, Indexes, Stored Procedures, Functions, Triggers, and Cursors.
- Applied exception handling and transaction management.
- Improved database programming and query optimization skills.
